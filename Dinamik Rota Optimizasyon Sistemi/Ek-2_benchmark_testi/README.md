Kurulum ve Çalıştırma:

Gerekli Kütüphanelerin Kurulumu:

Sistemin çalışması için gereken optimizasyon kütüphanelerini kurmak için terminalde şu komutu çalıştırın:

pip install -r requirements.txt

Testi başlatmak için terminalde şu komutu çalıştırın:

python ana_test.py

Program çalıştıktan sonra klasörde "ana_test_sonucu.txt" dosyası oluşacaktır ve test sonucu buraya işlenecektir.
